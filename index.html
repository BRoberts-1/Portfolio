About

My name is Binyamin and I enjoy building software. I became interested in software development when I worked as an echocardiographer performing diagnostic ultrasounds of adult hearts. The software used on the machines interested me, and I wanted to know how it worked under the hood. I had many questions: How is the data organized when there is so much of it? How is this data rendered so quickly when there are so many changes constantly occuring in real time? What algorithms and processes are used to handle so much data all at once. My driving curiosity led me to immerse myself in learning about how to not only understand but create such software. My self-studies in software engineering have been fruitful and I continue to learn daily as I put my skills into practice to build real world applications. Let's create together.
